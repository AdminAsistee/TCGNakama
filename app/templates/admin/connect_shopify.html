How to get your token

  1. 1. Go to your Shopify Admin → Settings → Apps and sales channels
  2. 2. Click Develop apps → Select your custom app
  3. 3. Go to API credentials tab
  4. 4. Copy the Admin API access token (starts with shpat_)
